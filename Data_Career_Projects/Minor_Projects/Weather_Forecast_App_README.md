# Weather Forecast App

## Description
Fetch and visualize weather data using OpenWeatherMap API. Use basic ML to predict future weather trends.

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib/Seaborn
- Streamlit (if applicable)

## Features
- Data cleaning & preprocessing
- Model training & evaluation
- Visualization

## How to Run
```bash
python main.py
```
